# Muse Design System

Welcome to the **Muse Design System** documentation! This living design system embodies the Muse methodology for joyful software creation.

## What is Muse?

Muse is a comprehensive methodology that emphasizes:

- **Human-Centered Design First**: Start with full user journeys, not isolated tasks
- **Physics of Flow**: Work flows "just in time" with no handoff friction  
- **Context-Native Components**: Built directly in their live data environments
- **Real Data Over Mock Data**: Edge cases are discovered during development

Every component is mapped to `.feature` files that define:
- User acceptance criteria
- Edge case handling
- Accessibility requirements
- Visual regression testing
